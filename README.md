## 📁 Como baixar o projeto

```bash
# Clonar o repositório
$ git clone https://github.com/ElielAlves42/Tela-de-Login

# Entrar no diretório
$ cd Tela-de-Login

# instalar as dependências
$ yarn install

# Iniciar o projeto
$ yarn start
```
